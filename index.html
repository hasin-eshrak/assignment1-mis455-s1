HTML

HTML (Hypertext Markup Language) is a standard language for creating and structuring content on web pages. It uses tags to define elements like headings, paragraphs, links, and images, allowing browsers to render and display information on the internet.


CSS

CSS is a styling language used in web development to control the visual presentation of HTML elements. It defines layout, colors, fonts, and other design aspects, ensuring a consistent and attractive appearance of web pages.


Java Script

JavaScript is a versatile programming language used in web development to add interactivity and dynamic behavior to websites. It enables functions like form validation, animations, and real-time updates, enhancing user experiences on the web.
